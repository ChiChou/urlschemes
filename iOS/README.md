# iOS URL Schemes Enumeration

No jailbreak required. Depends on [ideviceinstaller](https://github.com/libimobiledevice/ideviceinstaller)

The script forwards command flags to ideviceinstaller.

* `--udid` explicitly specify device UDID
* `--system` for system app only
* `--user` for user installed apps only
* `--all` system apps and user installed apps
